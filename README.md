
# PHP Lambert W Function
Implementation of Lambert W function in PHP.

# Installation
```
git clone https://github.com/ammarfaizi2/php_lambertw;
cd php_lambertw;
phpize;
./configure;
make -j2;
```

# Run The Test
```
php -dextension=$(pwd)/modules/lambertw.so test.php
```
